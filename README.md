# Silentium Review / 静研录

Silentium Review / 静研录 是一个考研刷题错题复盘 PWA。它帮助你记录错题、标记错因、写一句复盘提示，并按复盘节奏回看。

当前版本状态：MVP 稳定版。

## 核心功能

- 错题录入：支持题目图片、科目、模块、题型、结果、错因、复盘一句话和备注。
- 今日复盘：根据本地复盘计划显示当天需要回看的错题。
- 错题列表：按科目、模块、错因和掌握状态筛选。
- 统计概览：查看本周刷题数、错题数、高频错因和薄弱模块。
- JSON 备份：导出/导入本地数据。
- 示例数据：在设置页生成或清除带 `isDemo: true` 标记的示例数据。

## 本地启动

```bash
npm install
npm run dev
```

常用验证命令：

```bash
npm test
npm run build
npm run check
```

Windows PowerShell 如遇到 `npm.ps1` 执行策略限制，可以使用：

```bash
npm.cmd test
npm.cmd run build
npm.cmd run check
```

### 本地一键启动

- 双击 `启动静研录本地版.bat`。
- 等浏览器自动打开本地页面。
- 关闭命令窗口即可停止本地服务。
- 如果端口变化，以命令窗口显示的 Local 地址为准。

## 数据存储

数据保存在当前浏览器的 IndexedDB 中。应用目前不依赖后端服务，刷新页面后数据仍会保留在同一浏览器环境里。

注意事项：不同设备、不同浏览器之间的数据不会自动同步。换设备、换浏览器、清缓存或重装系统前，一定要先导出 JSON 备份。

## 备份方式

在设置页使用：

- 导出 JSON：保存错题、图片、复盘记录、刷题记录和设置。
- 导入 JSON：可合并导入，也可清空后导入。

建议定期导出备份，尤其是在阶段复盘、考试前整理、换电脑或清理浏览器数据之前。

## AI 使用说明

- 在设置页填写 DeepSeek API Key。
- 可按需调整模型名，并使用“测试连接”确认当前配置可用。
- API Key 只保存在当前浏览器，不会写入代码。
- JSON 导出不会包含 API Key。
- AI 复盘建议只分析错题的文字字段，例如科目、模块、题型、结果、错因、复盘、备注和解法文字。
- AI 不分析图片，不上传题目截图。
- 图片 OCR 是后续功能，当前不支持。

## 暂不支持

- OCR 识别
- 云同步
- 登录
- 后端 API

## GitHub Pages 部署准备

项目暂未加入复杂 GitHub Actions。未来部署到 GitHub Pages 时建议：

- Vite 当前使用 `base: './'`，适合发布到 GitHub Pages 的项目子路径。
- `dist/` 是构建产物，默认不提交；推荐由部署流程生成并发布。
- PWA manifest 使用相对路径，图标位于 `public/icons/`，构建后会复制到 `dist/icons/`。
- 如未来改为自定义域名或根路径部署，可以再把 `vite.config.ts` 中的 `base` 调整为 `/`。
